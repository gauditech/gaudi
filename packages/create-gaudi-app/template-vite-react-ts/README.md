# Gaudi demo project

Demo project with Gaudi server and Vite+React+TS frontend.

### Build

```
# entire project
npm run build

# client
npm run build:client

# gaudi server
npm run build:server:api

# server hooks
npm run build:server:hooks
```

### Dev mode

```
# entire project
npm run dev
```

### Lint

```
# entire project
npm run test

# client
npm run test:client

# gaudi server
npm run test:server
```

### Clean

```
npm run clean
```
